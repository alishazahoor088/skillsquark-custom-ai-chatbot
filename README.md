# Custom AI Chatbot

A custom AI chatbot built using Python, LangChain, FAISS, and Google Gemini.

## Features

- Upload PDF documents
- Extract document content
- Create vector embeddings
- Ask questions about uploaded PDFs
- AI-powered responses using Gemini

## Technologies Used

- Python
- Streamlit
- LangChain
- FAISS
- Google Gemini
- PyPDF

## Project Structure

- app.py
- chatbot.py
- pdf_loader.py
- vector_store.py
- requirements.txt
- .env
